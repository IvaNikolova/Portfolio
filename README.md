## Overview
This project is my **personal portfolio website**, designed to showcase my work in **data science, frontend development, and interactive applications**.  

The website presents selected projects, technical skills, and background information in a clean and responsive interface. It serves as a central hub where recruiters and collaborators can explore my work and learn more about my experience.

**Live website:** [Portfolio Website](https://ivanikolova.vercel.app)

---

## Featured Projects
The portfolio highlights several projects across data science and web development, including:

### 1. Madrid Bike-Sharing Simulation
A **data science and reinforcement learning project** that simulates bike-sharing activity in Madrid using historical trip data and a multi-agent Deep Q-Network (DQN) model.

### 2. FilmVault
A **movie discovery and wishlist platform** built with React and Firebase that allows users to browse movies, view details, and manage personalized watchlists.

### 3. Calendar & To-Do Desktop App
A **calendar-based productivity application** designed for flexible task scheduling and event planning.

## Run Locally
### Clone the repository
- `git clone https://github.com/IvaNikolova/your-portfolio-repo.git`
- `cd your-portfolio-repo`

### Install dependencies
- `npm install`

### Start development server
- `npm run dev`
Open the app in your browser at: `http://localhost:5173`
